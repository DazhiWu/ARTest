# 地下管网 AR 可视化系统 - 快速入门指南

## 🚀 快速开始

### 前置要求
- 现代移动设备（建议 Android 8.0+ 或 iOS 14.0+）
- HTTPS 连接或 localhost 开发环境
- 摄像头、位置、运动传感器权限

### 5 分钟快速上手

#### 1️⃣ 启动应用
```
在浏览器中打开 index.html
```

#### 2️⃣ 授权权限
- 允许摄像头访问
- 允许位置访问
- 允许运动传感器访问（iOS 需要）

#### 3️⃣ 启动地面检测
- 点击右侧面板的 "工具" 标签
- 点击 "🚀 智能地面检测" 按钮
- 系统将自动选择最佳检测模式

#### 4️⃣ 校准地面
**WebXR 模式（推荐）**:
- 缓慢移动设备，扫描周围地面
- 等待绿色网格出现
- 系统自动对齐管线

**手动模式**:
- 直接点击屏幕上的真实地面位置
- 绿色标记点出现
- 管线自动对齐

#### 5️⃣ 查看效果
- 摄像头画面中管线贴合真实地面
- 移动设备，管线保持与地面相对位置
- 使用控制面板调整参数

## 🎛️ 控制面板说明

### 图层控制
- 给水管线（蓝色）
- 排水管线（绿色）
- 电力管线（黄色）
- 燃气管线（红色）

### 深度控制
- 最小埋深：过滤浅于指定深度的管线
- 最大埋深：过滤深于指定深度的管线

### 工具面板
- **虚拟升降**: 调整观察高度
- **模型缩放**: 调整管线尺寸
- **快速视角**: 快速切换地面/俯视/侧视
- **智能地面检测**: 启动自动检测
- **手动地面校准**: 手动点击校准
- **位置校准**: 微调管线位置
- **重置视角**: 恢复默认设置

## 💡 使用技巧

### 获得最佳检测效果
1. **光线充足**: 避免暗光环境
2. **纹理丰富**: 选择有纹理的地面（地砖、水泥地）
3. **缓慢移动**: 让 AR 系统有时间扫描环境
4. **水平地面**: 优先选择平坦的水平地面

### 常见问题解决

**Q: 检测不到地面？**
- 检查光线条件
- 尝试不同的地面区域
- 使用手动校准模式

**Q: 管线位置不对？**
- 点击 "清除锚点" 重新校准
- 使用 "位置校准" 微调
- 检查埋深设置是否正确

**Q: 画面很卡？**
- 关闭不必要的后台应用
- 使用手动模式替代 WebXR
- 降低屏幕亮度

**Q: 管线漂浮在空中？**
- 重新校准地面位置
- 调整埋深参数
- 检查 GPS 定位是否准确

## 🔬 高级功能

### 加载真实管线数据
1. 将 GeoJSONL 文件放入 `public/` 目录
2. 点击 "加载真实数据" 按钮
3. 系统自动转换坐标并渲染

### 手动定位模式
1. 点击 "手动定位" 按钮
2. 输入经纬度坐标
3. 调整朝向角度
4. 应用并查看效果

### 距离测量
1. 点击 "距离测量" 按钮
2. 点击屏幕选择起点
3. 点击屏幕选择终点
4. 查看测量结果

## 📊 状态指示

- **GPS**: 定位状态
- **罗盘**: 方向传感器状态
- **锚点**: 空间锚定状态

### 状态颜色
- 🟢 绿色: 正常工作
- 🟡 黄色: 等待/校准中
- 🔴 红色: 错误/不可用
